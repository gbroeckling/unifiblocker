# Code of Conduct

## Our Pledge

We want UniFi Blocker to be a welcoming project for everyone — regardless of experience level, background, or how they write their code.

## Expected Behavior

- Be kind and constructive in issues, PRs, and discussions
- Assume good intent — most misunderstandings are just that
- Give helpful feedback, not just "this is wrong"
- Respect that people have different setups, hardware, and skill levels

## Unacceptable Behavior

- Personal attacks, insults, or dismissive language
- Harassment of any kind
- Posting others' private information without consent
- Trolling or deliberately disruptive behavior

## Enforcement

If someone violates this code of conduct, the maintainer may remove comments, close issues, or block users as needed. For serious concerns, email the maintainer directly through their GitHub profile.

## Scope

This applies to all project spaces — GitHub issues, pull requests, discussions, and any other communication channel associated with UniFi Blocker.
